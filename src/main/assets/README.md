# Pasta de Assets do MediaPipe

Esta pasta contém os arquivos de recursos para o seu projeto Android, incluindo modelos de Machine Learning do MediaPipe.

## Modelo FaceLandmarker

Para que a detecção de pontos faciais funcione corretamento, o arquivo do modelo `face_landmarker.task` deve ser colocado diretamente nesta pasta (`src/main/assets/`).

### Como obter o modelo:
1. Faça o download do arquivo de modelo oficial clicando no link abaixo:
   [face_landmarker.task (Google APIs Storage)](https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task)
2. Salve o arquivo baixado como `face_landmarker.task` nesta pasta:
   `/home/kiruma/AndroidStudioProjects/AplicationBeta01/app/src/main/assets/face_landmarker.task`
